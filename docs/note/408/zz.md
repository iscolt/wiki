### 计算机的基本组成

#### 冯·诺依曼计算机特点

> 存储程序控制

- 计算机由`运算器、控制器、存储器、输入、输出设备`五大部件组成
- 指令和数据以同等地位放于存储器内，按地址寻访
- 指令和数据都用二进制表示
- 指令由操作码和地址码组成，操作码是操作的性质，地址码既操作数据存放的位置
- 指令在存储器内按顺序存放
- 机器以运算器为中学

**运算器（ALU）**：算术与逻辑运算

**控制器（CU）**：控制、指挥程序和数据的输入、运行及结果

**存储器**：存放数据和程序

> 现代计算机可认为由三大部分组成：CPU、I/O设备、主存
>
> CPU + 主存 = 主机
>
> I/O设备 = 外部设备
